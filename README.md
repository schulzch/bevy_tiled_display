# Bevy Tiled Display

**Highly experimental and work in progress. Use at your own risk!**

This repository contains a Bevy plugin for a large multi-machine tiled display support.

## Quick start

Build the project:

```sh
cargo build
```

Run tests:

```sh
cargo test
```

## Example

```sh
cargo run --example demo
```
